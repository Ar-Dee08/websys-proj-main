git commands:
"git remove -v" to check the origin and main repos
"git fetch upstream" fetch the latest changes from upstream
"git checkout main" local main branch
"git merge upstream/main" merge the changes from the upstream to main
"git push origin main" push the updates to your remote fork
"git add <file name>" resolving conflicts, stage the changes
"git commit -m "Resolved Merge Conflicts"" commit the changes
"git push origin main" push the resolved origin main
"git pull origin main" synch your main with the remote repo
